**Shell** (оболочка) и **Bash** — это программы-посредники между пользователем и операционной системой.

### **Shell (оболочка)** - это КОНЦЕПЦИЯ

**Что делает:** Принимает текстовые команды от пользователя → передаёт их ядру ОС → показывает результат.

**Простая аналогия:** Переводчик-помощник:

- Вы говорите команду на "человеческом" языке

- Shell переводит её на "машинный" язык ядра

- Показывает вам ответ

**Примеры shell:** bash, zsh, fish, csh, ksh

---
### **Какой shell у вас?**

```bash
echo $SHELL    # Покажет вашу текущую оболочку
which bash     # Где находится bash
```

Shell — это "рабочее окружение" в командной строке, а Bash — самый популярный и мощный вариант этого окружения, ставший стандартом де-факто в Linux-мире.
