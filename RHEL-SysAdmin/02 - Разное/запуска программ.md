## **Как интерпретатор запускает программы?**

Когда вы вводите команду в bash, происходит:

```bash
# Пример: вы вводите
nginx
```

**Что делает bash:**

1. **Поиск:** Ищет файл `nginx` в папках из переменной `$PATH` (/usr/bin, /bin и т.д.)

2. **Проверка прав:** Смотрит, есть ли у вас право на выполнение этого файла

3. **Создание процесса:** Вызывает системную функцию `fork()` для создания копии себя

4. **Запуск:** В копии вызывает `exec()`, которая заменяет код bash на код программы nginx

5. **Ожидание:** Ждёт завершения (или уходит в фон с `&`)
