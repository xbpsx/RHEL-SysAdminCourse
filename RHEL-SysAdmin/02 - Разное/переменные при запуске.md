**Примеры на серверах:**

```bash
# Переменные окружения для приложения
export DATABASE_URL="postgresql://user:pass@db-server:5432/app"
export REDIS_HOST="redis-server"
export DEBUG="false"

# Запуск приложения с этими переменными
python app.py

# Или прямо в одной строке:
DATABASE_URL="..." REDIS_HOST="..." python app.py
```

**Что это даёт:**

- ✅ **Безопасность:** Пароли не в коде, а в переменных окружения

- ✅ **Гибкость:** Один и тот же код работает в dev/stage/prod окружениях

- ✅ **Удобство:** Легко менять настройки без пересборки приложения
