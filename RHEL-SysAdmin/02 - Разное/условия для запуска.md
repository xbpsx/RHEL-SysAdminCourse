**Примеры условий и зачем они нужны:**

```bash
# Запускать только если сервис не уже запущен
if ! systemctl is-active --quiet nginx; then
    systemctl start nginx
fi

# Запускать только если есть достаточно свободной памяти
if [ $(free -m | awk '/Mem:/ {print $4}') -gt 512 ]; then
    java -jar app.jar
fi

# Запускать с разными параметрами в зависимости от времени суток
if [ $(date +%H) -ge 8 ] && [ $(date +%H) -lt 20 ]; then
    node app.js --mode=day
else
    node app.js --mode=night
fi
```

**Зачем:** Чтобы избежать ошибок, оптимизировать ресурсы и адаптировать поведение к окружению.
