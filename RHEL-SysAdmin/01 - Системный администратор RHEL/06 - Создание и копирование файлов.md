Для созданию файлов, можно использовать `touch file`, основная задача `touch` обновить время доступа к файлу. Если файл существует и приминеть на него `touch`, то с ним ничего не случится, только обновится информация в ноде, а если файла не существует то он создаться.

Одна из фишек командной строки, возможность работать с [[регулярными выражениями]] , например мы омжем написать `touch file{5..15}`

![[screenshot-20251113-083605.png]]

То же работает с `mkdir dir_{a..f}`

![[screenshot-20251113-083759.png]]

Что бы создать нивидимые файлы и дириектории, сначала нужно указать .
`mkdir .wallpapers`  или `touch .file` 

Что бы увидеть скрытые файлы и директории `ls -a` с ключом а, в файловом менеджере пишем `ctrl + h` , обычно скрытые файлы нужны всяким программам для хранение настроек в домашней директории пользователя, но не только.

### Копирование

**Для копирования** используется команда `cp` , чтобы скопировать файл в текущей директории нужно написать `cp filename newfile`. Многие команды сами могут показать, что происходит вовремя работы, для этого используется опция `-v`, например `cp -v filename newfile`, 

![[screenshot-20251113-084637.png]]

Тут мы видим, что файл скопировался. Чтобы **скорпировать файл в  другую директорию**

```bash
[darker@archlinux .wallpapers]$ cp -v arch.png /home/darker/Memory
'arch.png' -> '/home/darker/Memory/arch.png'
```

Скопировать файл и переименовать

```bash
[darker@archlinux .wallpapers]$ cp -v arch.png hello.png
'arch.png' -> 'hello.png'
[darker@archlinux .wallpapers]$ ls
arch.png  hello.png  home.png
```

Скопировать несколько файлов в папку

```bash
[darker@archlinux .wallpapers]$ cp -v arch.png home.png /home/darker/Memory
'arch.png' -> '/home/darker/Memory/arch.png'
'home.png' -> '/home/darker/Memory/home.png'
```

**`-i`** (**интерактивный режим**):

Спросить подтверждение перед перезаписью.

Опция `-i` срабатывает только при **перезаписи существующих файлов**. Если в целевой папке нет файлов с такими же именами, подтверждение не запрашивается.

```bash
cp -i -v -r Pictures /home/darker/Memory/
```

**`-u`** (**обновление):**

Копировать, только если источник новее или отсутствует в назначении.

```bash
cp -u file.txt /path/to/destination/
```

**`-p`**:

Сохранить атрибуты (время изменения, права доступа).

```bash
cp -p file.txt /path/to/destination/
```

**`-l`** (жесткие ссылки) или **`-s`** (символические ссылки):

Создать ссылки вместо копирования файлов.

```bash
cp -s file.txt link_to_file.txt  # Символическая ссылка
```
