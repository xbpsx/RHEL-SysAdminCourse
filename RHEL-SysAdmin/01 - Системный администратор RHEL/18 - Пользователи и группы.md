Наконец-то пришло время в файле 

```bash
nano /etc/passwd
```

разобраться, что же тут написано. Каждая строчка, содержит информацию о каком-то пользователе. 

```bash
  1   2   3    4    5     6          7
xbpsx:x:1000:1000:xbpsx:/home/xbpsx:/usr/bin/fish
```

`:` - в данном файле выступает в роли разделителя, специального символа который делит строку на столбцы. 

1. **В первом** столбце у нас `xbpsx` логин пользователя.

2. **Во втором** `x` очень давно тут хранились пароли пользователей в хэшированном виде. При хэширвоание к паролю добавляют рандомные символы, называемые **солью**, благодаря чему, два одинковых пароля после хэширования буду выглядить по разному. 

Пароли из `etc/passwd` перенесли в другой файл в 

```bash
sudo nano /etc/shadow
```

```text
xbpsx:$6$fF7PV/e0H$XQdRwZCfLO8pRjKEvVOBvo96X2tuXqI6dU8TDB80gLR6/54.4To.TMV06Lij63p/5BLZN8kz7ZOwNBeHsoyGs.:20441:0:99999:7:::
```

3. **Третрий столбик** , уникальный индефикатор пользователя `1000` userid, uid. Очень многое строится вокруг `uid`, например у **root** пользователя всегда `uid 0`. Обычно для необходимых сервисных пользователей `uid` назначаются ниже < 100. А для дополнительных сервисных пользователей до 999. А с 1000 начинаются `uid` обычных пользлвателей.

4. **Четверный столбик**, идет индефикатор группы `groupd id`. У каждого пользователя есть одна основная группа и у каждой группы есть свой уникальный индентификатор. А для самих групп есть файл

```bash
sudo nano /etc/group
```

```
root:x:0:
bin:x:1:
sys:x:2:
kmem:x:3:xbpsx
wheel:x:4:xbpsx
tty:x:5:xbpsx
tape:x:6:xbpsx
daemon:x:7:xbpsx
floppy:x:8:xbpsx
disk:x:9:xbpsx,libvirt
lp:x:10:
dialout:x:11:xbpsx
...
```

5. В этом столбце у меня написано `xbpsx` . Тут пишут небльшой комментарий о пользователе, полное имя пользователя, телефонный номер и тд.

6. Домашняя директория пользователя, у меня `/home/xbpsx`
7. Последний столбик, **shell** оболочка пользователя, у меня стоит `/usr/bin/fish` 
У сервисных пользователей вместо оболчки указан `/sbin/nologin` , если кто-то попытается залогингится этими пользователями выйдет текст

```bash
This account is currently not available
```

 В некоторывх случаев можно увидеть `/bin/false` тоже не дает залогиниться.
